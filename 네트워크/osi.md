# OSI 7계층
## 1. 물리 계층 (Physical Layer)
* 전기 신호를 이용하여 비트스트림을 전송하는 계층이다.
* 물리계층은 하드웨어적으로 구현되어있다.

## 2. 데이터 링크 계층 (Data Link Layer)
* 물리계층을 통해 전달되는 데이터의 흐름과 오류를 관리해서 같은 네트워크 안의
여러 컴퓨터간 안전하게 데이터를 전달할수 있도록 하는 계층이다.
* 전송 단위는 프레임이다.
* 데이터링크 계층은 하드웨어적으로 구현되어있다.

## 3. 네트워크 계층 (Network Layer)
* 라우팅을 통해 안전하고 빠르게 다른 네트워크의 컴퓨터로 데이터를 전송하는 계층이다.
* 전송 단위는 패킷이다.
* 네트워크 계층은 운영체제 커널에 소프트웨어적으로 구현되어있다.

## 4. 전송 계층 (Transport Layer)
* 양 끝단의 사용자들이 신뢰성있는 데이터를 주고 받을 수 있게 해주는 계층이다.
* 전송 계층은 운영체제 커널에 소프트웨어적으로 구현되어있다.
### 종류
* TCP
* UDP
* DNS
* DHCP

## 5. 응용 계층 (Application Layer)
* 프로세스와 관계하여 응용서비스를 수행하는 계층이다.
### 종류
* HTTP
